# Credit Risk Prediction using Machine Learning

## Project Overview
This project builds a Credit Risk Prediction model using Logistic Regression.
The goal is to predict whether a loan application will be approved or rejected based on applicant data.

## Technologies Used
- Python
- Pandas
- Seaborn & Matplotlib
- Scikit-learn

## Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. Encoding Categorical Variables
4. Feature Scaling
5. Model Training (Logistic Regression)
6. Model Evaluation

## Model Performance
- Accuracy Score
- Confusion Matrix

## How to Run
```bash
pip install -r requirements.txt
python src/credit_risk_model.py
